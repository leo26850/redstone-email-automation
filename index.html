Redstone Manufacturing · RED-429

Lead-Form Auto-Reply Automation

Draft for review Prepared by KAMG · 2026-07-14 Client draft due: Fri 2026-07-17

In one sentence: every enterprise-domain lead-form submission automatically receives Manuel's personal reply within ~5 minutes — from his real mailbox, with Eric and Alec CC'd, the submission quoted, a booking link included, and the email logged in HubSpot — while free-mail submissions are silently skipped and any lead reply that misses Alec is auto-forwarded to him.

1The brief

From Eric's email of 2026-07-09. Eight requirements:

  1. Qualify the lead — auto-reply only for enterprise domains (no gmail etc.)
  2. Reply from Manuel's email account
  3. Manuel's exact template/wording from the Teledyne thread, with Alec's calendar link added at the end
  4. CC Eric and Alec on the auto-reply
  5. Replies that go only to Manuel auto-forward to Alec (if the lead doesn't reply-all)
  6. Email logged in HubSpot for the lead
  7. Include the lead-form submission in the email, like Manuel did
  8. Subject line: "XXXX with Redstone Manufacturing", where XXXX = the manufacturing process from the form

2Architecture decision

One Google Apps Script project owned by manuel@redstonemanufacturing.com. No servers, no paid middleware, no website changes.

  • Runs as Manuel — sends are genuine emails from his mailbox: real threading, real CCs, lands in his Sent folder. Requirement 2 comes for free.
  • The notification email is the trigger — form submissions already land in Gmail, so the script polls the inbox every 5 minutes. No webhooks, no HubSpot-tier dependency, no website changes.
  • Reply-forwarding needs mailbox access anyway — Zapier/Make and HubSpot workflows handle "forward replies in this thread" poorly; a Gmail-side script does it natively.
  • Zero marginal cost — Workspace quotas allow ~1,500 sends/day; lead volume is nowhere near.

3System flow

Form submission (website) ──► also lands in HubSpot (as today)
        └──► notification email lands in Manuel's Gmail
                        │
        Time-driven trigger, every 5 min → main()
                        │
   ┌── processNewLeads_()
   │   1. Gmail search: unprocessed "Tell us about your project" emails
   │   2. Parse fields: Work Email, Name, Company, Process, Stage, Reason, Date, Phone
   │   3. Free-mail domain?  → label lead-auto/skipped, STOP (no reply ever)
   │   4. Send as Manuel:
   │        To:      lead's work email
   │        CC:      eric@ + alec@redstonemanufacturing.com
   │        Subject: "{Process} with Redstone Manufacturing"
   │        Body:    Manuel's verbatim template + calendar link
   │                 + signature + form submission quoted underneath
   │   5. Log email engagement on the HubSpot contact (private-app API)
   │   6. Labels: notification → lead-auto/processed, sent thread → lead-auto/sent
   │
   └── forwardLoneReplies_()
       Scan lead-auto/sent threads (60-day window).
       Message FROM the lead where alec@ is NOT on To/Cc
       → forward to Alec, dedup-tracked so nothing forwards twice.

State machine & audit trail = Gmail labels: lead-auto/processed, lead-auto/skipped, lead-auto/sent, lead-auto/error. Failures label the thread error and never auto-retry — everything is inspectable straight from Manuel's inbox.

/**
 * Redstone Manufacturing — lead-form auto-reply automation (RED-429)
 *
 * Runs as an Apps Script project OWNED BY manuel@redstonemanufacturing.com.
 * Trigger: time-driven, every 5 minutes → main()
 *
 * Flow:
 *  1. Find new lead-form notification emails in this inbox.
 *  2. Parse the form fields (work email, name, process, …).
 *  3. If the lead's email is on an enterprise domain (not free-mail):
 *     send Manuel's templated reply to the lead, CC Eric + Alec,
 *     subject "{Process} with Redstone Manufacturing",
 *     with the form submission quoted at the bottom and Alec's
 *     calendar link appended.
 *  4. Log the sent email against the contact in HubSpot (private app).
 *  5. Separately: watch auto-reply threads for lead replies that did NOT
 *     include Alec (no reply-all) and forward those to Alec.
 *
 * Setup: see README.md. Run setup() once after setting Script Properties.
 */

var CONFIG = {
  // People
  cc: 'eric@redstonemanufacturing.com, alec@redstonemanufacturing.com',
  forwardTo: 'alec@redstonemanufacturing.com',
  companyDomain: 'redstonemanufacturing.com',

  // OPEN ITEM: Alec's meeting-scheduler link (set as Script Property ALEC_CALENDAR_LINK)
  calendarLinkFallback: '',

  // Gmail search that matches raw lead-form notification emails.
  // TODO: tighten with the real sender once confirmed, e.g. 'from:forms@… '
  notificationQuery: 'subject:"Tell us about your project" -label:lead-auto/processed -label:lead-auto/skipped -label:lead-auto/error newer_than:3d',

  labels: {
    processed: 'lead-auto/processed',
    skipped: 'lead-auto/skipped',   // free-mail / unqualified
    sent: 'lead-auto/sent',         // threads where we sent the auto-reply
    error: 'lead-auto/error',
  },

  freemailDomains: [
    'gmail.com', 'googlemail.com', 'yahoo.com', 'ymail.com', 'hotmail.com',
    'outlook.com', 'live.com', 'msn.com', 'icloud.com', 'me.com', 'mac.com',
    'aol.com', 'proton.me', 'protonmail.com', 'gmx.com', 'gmx.net',
    'zoho.com', 'mail.com', 'yandex.com', 'yandex.ru', 'qq.com', '163.com',
    '126.com', 'comcast.net', 'verizon.net', 'att.net', 'sbcglobal.net',
  ],
};

// Field labels as they appear in the lead-form notification email.
var FORM_FIELDS = [
  'Current Stage of Production',
  'Type of Manufacturing Process',
  'Main Reason for Requesting Estimate',
  'Target Date of First Shipment',
  'Name',
  'Company',
  'Work Email',
  'Phone Number',
];

// ---------------------------------------------------------------- main

function main() {
  processNewLeads_();
  forwardLoneReplies_();
}

/** One-time setup: creates labels and the 5-minute trigger. */
function setup() {
  Object.keys(CONFIG.labels).forEach(function (k) {
    getOrCreateLabel_(CONFIG.labels[k]);
  });
  ScriptApp.getProjectTriggers().forEach(function (t) {
    if (t.getHandlerFunction() === 'main') ScriptApp.deleteTrigger(t);
  });
  ScriptApp.newTrigger('main').timeBased().everyMinutes(5).create();
  Logger.log('Setup complete: labels created, 5-minute trigger installed.');
}

// ------------------------------------------------- 1. new-lead pipeline

function processNewLeads_() {
  var threads = GmailApp.search(CONFIG.notificationQuery, 0, 20);
  threads.forEach(function (thread) {
    try {
      var msg = thread.getMessages()[0];
      var body = msg.getPlainBody();
      var lead = parseLeadForm_(body);

      if (!lead.email) {
        throw new Error('Could not parse Work Email from notification ' + msg.getId());
      }

      if (!isEnterpriseDomain_(lead.email)) {
        thread.addLabel(getOrCreateLabel_(CONFIG.labels.skipped));
        thread.addLabel(getOrCreateLabel_(CONFIG.labels.processed));
        Logger.log('Skipped (free-mail): ' + lead.email);
        return;
      }

      var sent = sendAutoReply_(lead, body, msg.getDate());
      logToHubSpot_(lead, sent.subject, sent.textBody);

      thread.addLabel(getOrCreateLabel_(CONFIG.labels.processed));
      Logger.log('Auto-reply sent to ' + lead.email + ' — "' + sent.subject + '"');
    } catch (e) {
      thread.addLabel(getOrCreateLabel_(CONFIG.labels.error));
      thread.addLabel(getOrCreateLabel_(CONFIG.labels.processed));
      Logger.log('ERROR on thread ' + thread.getId() + ': ' + e.message);
    }
  });
}

function sendAutoReply_(lead, rawFormBody, formDate) {
  var process = lead.fields['Type of Manufacturing Process'] || 'Manufacturing';
  var subject = process + ' with Redstone Manufacturing';
  var firstName = firstName_(lead.fields['Name']);
  var calendarLink = scriptProp_('ALEC_CALENDAR_LINK') || CONFIG.calendarLinkFallback;

  var formBlock = extractFormBlock_(rawFormBody);
  var dateStr = Utilities.formatDate(formDate, Session.getScriptTimeZone(), "EEE, MMM d, yyyy 'at' h:mm a");

  var textBody =
    'Hi ' + firstName + ',\n\n' +
    "Thanks for reaching out. I'd like to schedule a time to chat to learn more.\n\n" +
    'When would be a good time to connect with you over a video call?\n\n' +
    (calendarLink ? 'You can also book a time directly here: ' + calendarLink + '\n\n' : '') +
    'Manuel Ayala | Account Executive\n' +
    'Redstone Manufacturing\n' +
    'Direct: 330-632-3292\n' +
    'redstonemanufacturing.com\n\n' +
    'On ' + dateStr + ':\n\n' +
    quoteBlock_(formBlock);

  var htmlBody =
    '<p>Hi ' + escapeHtml_(firstName) + ',</p>' +
    "<p>Thanks for reaching out. I'd like to schedule a time to chat to learn more.</p>" +
    '<p>When would be a good time to connect with you over a video call?</p>' +
    (calendarLink
      ? '<p>You can also book a time directly here: <a href="' + calendarLink + '">' + calendarLink + '</a></p>'
      : '') +
    '<p><b>Manuel Ayala</b> | Account Executive<br>' +
    'Redstone Manufacturing<br>' +
    'Direct: 330-632-3292<br>' +
    '<a href="https://redstonemanufacturing.com">redstonemanufacturing.com</a></p>' +
    '<p>On ' + escapeHtml_(dateStr) + ':</p>' +
    '<blockquote style="margin:0 0 0 .8ex;border-left:1px solid #ccc;padding-left:1ex;color:#555;">' +
    escapeHtml_(formBlock).replace(/\n/g, '<br>') +
    '</blockquote>';

  GmailApp.sendEmail(lead.email, subject, textBody, {
    cc: CONFIG.cc,
    htmlBody: htmlBody,
    name: 'Manuel Ayala',
  });

  // Label the sent thread so the reply-forwarder can watch it.
  Utilities.sleep(2000);
  var sentThreads = GmailApp.search('in:sent to:' + lead.email + ' subject:"' + subject + '"', 0, 1);
  if (sentThreads.length) sentThreads[0].addLabel(getOrCreateLabel_(CONFIG.labels.sent));

  return { subject: subject, textBody: textBody };
}

// --------------------------------------------- 2. reply-forward watcher

/**
 * For every auto-reply thread, forward to Alec any message from the lead
 * where Alec is not on To/Cc (i.e. the lead replied only to Manuel).
 */
function forwardLoneReplies_() {
  var threads = GmailApp.search('label:' + CONFIG.labels.sent + ' newer_than:60d', 0, 50);
  var store = PropertiesService.getScriptProperties();
  var forwarded = JSON.parse(store.getProperty('FORWARDED_IDS') || '[]');

  threads.forEach(function (thread) {
    thread.getMessages().forEach(function (msg) {
      var id = msg.getId();
      if (forwarded.indexOf(id) !== -1) return;

      var from = (msg.getFrom() || '').toLowerCase();
      if (from.indexOf('@' + CONFIG.companyDomain) !== -1) return; // our own mail

      var recipients = ((msg.getTo() || '') + ',' + (msg.getCc() || '')).toLowerCase();
      if (recipients.indexOf(CONFIG.forwardTo.toLowerCase()) !== -1) return; // Alec already on it

      msg.forward(CONFIG.forwardTo, {
        subject: 'Fwd: ' + msg.getSubject(),
      });
      forwarded.push(id);
      Logger.log('Forwarded lone reply ' + id + ' to ' + CONFIG.forwardTo);
    });
  });

  // keep the store bounded
  if (forwarded.length > 2000) forwarded = forwarded.slice(-1000);
  store.setProperty('FORWARDED_IDS', JSON.stringify(forwarded));
}

// ------------------------------------------------------ 3. HubSpot log

/**
 * Logs the sent email as an Email engagement on the HubSpot contact.
 * Requires Script Property HUBSPOT_TOKEN (private-app token with scopes:
 * crm.objects.contacts.read, crm.objects.contacts.write, sales-email-read
 * is NOT needed — engagement writes use crm.objects.* email scope).
 */
function logToHubSpot_(lead, subject, textBody) {
  var token = scriptProp_('HUBSPOT_TOKEN');
  if (!token) {
    Logger.log('HUBSPOT_TOKEN not set — skipping HubSpot logging');
    return;
  }

  var contactId = findOrCreateContact_(lead, token);

  var payload = {
    properties: {
      hs_timestamp: new Date().toISOString(),
      hs_email_direction: 'EMAIL',
      hs_email_status: 'SENT',
      hs_email_subject: subject,
      hs_email_text: textBody,
    },
    associations: [{
      to: { id: contactId },
      types: [{ associationCategory: 'HUBSPOT_DEFINED', associationTypeId: 198 }], // email → contact
    }],
  };

  hubspot_('https://api.hubapi.com/crm/v3/objects/emails', 'post', token, payload);
}

function findOrCreateContact_(lead, token) {
  var search = hubspot_('https://api.hubapi.com/crm/v3/objects/contacts/search', 'post', token, {
    filterGroups: [{ filters: [{ propertyName: 'email', operator: 'EQ', value: lead.email }] }],
    properties: ['email'],
    limit: 1,
  });
  if (search.results && search.results.length) return search.results[0].id;

  // Form should have created the contact already; create as fallback.
  var name = (lead.fields['Name'] || '').trim().split(/\s+/);
  var created = hubspot_('https://api.hubapi.com/crm/v3/objects/contacts', 'post', token, {
    properties: {
      email: lead.email,
      firstname: name[0] || '',
      lastname: name.slice(1).join(' '),
      company: lead.fields['Company'] || '',
      phone: lead.fields['Phone Number'] || '',
    },
  });
  return created.id;
}

function hubspot_(url, method, token, payload) {
  var resp = UrlFetchApp.fetch(url, {
    method: method,
    contentType: 'application/json',
    headers: { Authorization: 'Bearer ' + token },
    payload: JSON.stringify(payload),
    muteHttpExceptions: true,
  });
  var code = resp.getResponseCode();
  if (code >= 300) throw new Error('HubSpot ' + code + ' on ' + url + ': ' + resp.getContentText().slice(0, 300));
  return JSON.parse(resp.getContentText() || '{}');
}

// -------------------------------------------------------- parsing utils

function parseLeadForm_(text) {
  var fields = extractFields_(text, FORM_FIELDS);
  var emailRaw = fields['Work Email'] || '';
  var emailMatch = emailRaw.replace(/\s/g, '').match(/[A-Za-z0-9._%+\-]+@[A-Za-z0-9.\-]+\.[A-Za-z]{2,}/);
  return {
    email: emailMatch ? emailMatch[0].toLowerCase() : null,
    fields: fields,
  };
}

/** Slice the text between each known label and the next one. */
function extractFields_(text, labels) {
  var found = [];
  labels.forEach(function (label) {
    var re = new RegExp('\\*?\\s*' + escapeRe_(label) + '\\s*\\*?\\s*:?', 'i');
    var m = re.exec(text);
    if (m) found.push({ label: label, start: m.index, end: m.index + m[0].length });
  });
  found.sort(function (a, b) { return a.start - b.start; });

  var out = {};
  found.forEach(function (f, i) {
    var next = found[i + 1];
    var val = text.slice(f.end, next ? next.start : text.length);
    val = val.replace(/Contact Information/i, ' ')
             .replace(/Tell us about your project/i, ' ')
             .replace(/[*​]/g, ' ')
             .replace(/^\s*[-•]\s*/gm, '')
             .replace(/\s+/g, ' ')
             .trim();
    out[f.label] = val;
  });
  return out;
}

/** The form portion of the notification body, for quoting in the reply. */
function extractFormBlock_(text) {
  var start = text.search(/Tell us about your project/i);
  var block = start >= 0 ? text.slice(start) : text;
  return block.replace(/\r/g, '').replace(/\n{3,}/g, '\n\n').trim();
}

function isEnterpriseDomain_(email) {
  var domain = email.split('@')[1];
  return CONFIG.freemailDomains.indexOf(domain) === -1;
}

function firstName_(name) {
  var first = (name || '').trim().split(/\s+/)[0] || 'there';
  return first.charAt(0).toUpperCase() + first.slice(1);
}

function quoteBlock_(text) {
  return text.split('\n').map(function (l) { return '> ' + l; }).join('\n');
}

function getOrCreateLabel_(name) {
  return GmailApp.getUserLabelByName(name) || GmailApp.createLabel(name);
}

function scriptProp_(key) {
  return PropertiesService.getScriptProperties().getProperty(key);
}

function escapeRe_(s) {
  return s.replace(/[.*+?^${}()|[\]\\]/g, '\\$&');
}

function escapeHtml_(s) {
  return String(s).replace(/&/g, '&amp;').replace(/</g, '&lt;').replace(/>/g, '&gt;').replace(/"/g, '&quot;');
}

// ------------------------------------------------------------- testing

/** Dry run: parse the newest matching notification and log what WOULD be sent. */
function test_parseLatest() {
  var threads = GmailApp.search(CONFIG.notificationQuery.replace(/-label:\S+/g, ''), 0, 1);
  if (!threads.length) { Logger.log('No matching notification email found.'); return; }
  var body = threads[0].getMessages()[0].getPlainBody();
  var lead = parseLeadForm_(body);
  Logger.log('Parsed email: ' + lead.email);
  Logger.log('Enterprise: ' + (lead.email ? isEnterpriseDomain_(lead.email) : 'n/a'));
  Logger.log('Fields: ' + JSON.stringify(lead.fields, null, 2));
  Logger.log('Subject would be: ' + (lead.fields['Type of Manufacturing Process'] || 'Manufacturing') + ' with Redstone Manufacturing');
}
